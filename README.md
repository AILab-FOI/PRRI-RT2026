# PRRI-RT2026

A RayTracing based FPS game developed by students and the [Artificial Intelligence Laboratory](https://ai.foi.hr/) at the [University of Zagreb Faculty of Organization and Informatics](https://www.foi.unizg.hr/). The game is developed using Python and PyGame. More details available at [itch.io](https://ailab-foi.itch.io/prri-rt2026).
